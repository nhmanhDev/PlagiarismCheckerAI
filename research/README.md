# Research Files

This folder contains research scripts and experimental code.

## Files

- **`experiments.py`** - Benchmark experiments and evaluation
- **`research_implementation.py`** - Research evaluation framework
- **`research_extensions.py`** - Extended research methods
- **`benchmark_loader.py`** - Dataset loaders for benchmarking

## Usage

These files are for research and development purposes.  
The main application in `/backend` and `/frontend` doesn't depend on them.

To run experiments:
```bash
cd research
python experiments.py
```

**Note:** Requires additional dependencies. See each file for specific requirements.
